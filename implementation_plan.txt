2. ARCHITECTURE & TECHNOLOGY STACK
================================================================================

RECOMMENDED STACK:
  ┌────────────────────────────────────────────────────────┐
  │ Framework:    Vite + Vanilla JS (single-page app)      │
  │ Styling:      Tailwind CSS (matching existing config)   │
  │ Routing:      Custom hash-based router (lightweight)    │
  │ State:        Vanilla JS Store (localStorage-backed)    │
  │ Build Tool:   Vite                                      │
  │ Deployment:   Static hosting (Netlify/Vercel/GitHub)    │
  └────────────────────────────────────────────────────────┘

WHY THIS STACK:
  - The existing pages are pure HTML+Tailwind — no React/Vue overhead needed
  - Vite gives us fast dev server, module bundling, and HMR
  - localStorage for cart/wishlist persistence (no backend required for MVP)
  - Hash-based routing keeps it simple for static hosting

ALTERNATIVE (if user wants backend):
  - Next.js + API routes + PostgreSQL/Supabase
  - Would enable real auth, real payments (Stripe), real order management
